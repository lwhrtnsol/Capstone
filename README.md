## Capstone Artifacts

### Software Design and Engineering: Travlr Getaways

Travlr Getaways is a full-stack travel application built with Node.js, Express, MongoDB, Handlebars, and Angular.

- [Original Artifact](artifacts/travlr.zip)
- [Enhanced Artifact](artifacts/ChrisMitchell_CS499_MilestoneTwo_Travlr_Artifact.zip)
- [Enhancement Narrative](narratives/CS499%20Milestone%202%20Narrative.docx)

Key enhancements include schema validation, improved API error handling, environment-based configuration, JWT security improvements, and dynamic trip rendering.

### Algorithms and Data Structures: Project Two SSL Server

Project Two SSL Server is a Java Spring Boot application that uses HTTPS and SHA-256 hashing.

- [Original Artifact](artifacts/ChrisMitchell_SSLServer_Original_SourceCode.zip)
- [Enhanced Artifact](artifacts/ChrisMitchell_SSLServer_SourceCode.zip)
- [Enhancement Narrative](narratives/CS499%20Module%204%20Milestone%203.docx)

Key enhancements include separating hashing logic into a service class, validating input, returning structured JSON responses, improving byte-to-hexadecimal conversion, and using environment-based SSL configuration.

### Databases: ChrisMitchellInventory Android Application

ChrisMitchellInventory is an Android application that uses SQLite to manage user accounts and inventory records.

- [Original Artifact](artifacts/ChrisMitchellInventory.zip)
- [Enhanced Artifact](artifacts/ChrisMitchell_CS499_MilestoneFour_Database_Artifact.zip)
- [Enhancement Narrative](narratives/ChrisMitchell%20CS499%20Milestone%204%20Database%20Narrative.docx)

Key enhancements include user-specific inventory records, foreign key relationships, data validation, data-preserving migration logic, repository improvements, and secure password handling.
